# BrainBites
BrainBites delivers quick, engaging mini-games designed to sharpen your mind in minutes. Challenge yourself daily with puzzles, logic games, and skill-based activities. Learn, compete, and grow smarter—one bite at a time.

## ✨ Features

* Bite-sized brain games
* Daily challenges
* Cross-platform support (Android, iOS, Desktop)
* Built with Kotlin Multiplatform & Compose Multiplatform

## 🛠 Tech Stack

* Kotlin Multiplatform (KMP)
* Compose Multiplatform
* Gradle

## 📱 Platforms

* Android
* iOS
* Desktop (JVM)

## 🚀 Getting Started

### Run Android

```bash
./gradlew :composeApp:assembleDebug
```

### Run Desktop

```bash
./gradlew :composeApp:run
```

### Run iOS

Open the `iosApp` folder in Xcode and run the project.

## 📂 Project Structure

* `composeApp/` → Shared code for all platforms
* `iosApp/` → iOS entry point
